`saucectl` configurations are versioned and each version in turn supports a range of test framework versions.

| Cypress Version      | SauceCTL Config Version |
| -------------------- | ----------------------- |
| Cypress >= 10        | [v1](v1)                |
| Cypress <= 9         | [v1alpha](v1alpha)      |
